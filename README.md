# Basic C# Programs for Pitman Training

This repository contains C# practice projects created during my personal study at Pitman Training. Each project demonstrates fundamental C# programming concepts and progressively builds programming skills.

## 🎯 Purpose
Personal learning repository for mastering C# fundamentals through hands-on projects.

## 🛠️ Technologies
- **Language:** C# (.NET 10)
- **IDE:** Visual Studio Community 2026 (18.4.3)

## 📂 Projects

### 1. Hello World
**Location:** `Hello_World/`  
**Description:** Classic "Hello World" program - the first step in learning any programming language.  
**Concepts Covered:**
- Basic console output
- Program structure
- `Console.WriteLine()` method

---

### 2. myConsoleProject
**Location:** `myConsoleProject/`  
**Description:** Interactive console application that prompts for user input and provides personalized greeting.  
**Concepts Covered:**
- User input with `Console.ReadLine()`
- String variables
- String interpolation (`$"{variable}"`)
- Basic user interaction

---

### 3. Writing and Running Code in C# Assignment
**Location:** `Writing and Running Code in C# Assignment/`  
**Description:** Professional greeting application for Acme Accounting Systems demonstrating string formatting.  
**Concepts Covered:**
- Namespace organization
- String literals with escape sequences (`\n` for newlines)
- Quotation marks in strings (`\"`)
- Multi-line console output

---

## 🚀 How to Run

1. Clone this repository:
   ```bash
   git clone https://github.com/Thomas-B04/Basic_C_Sharp_Programs_For_Pitman_Training.git
   ```

2. Open the solution in Visual Studio 2026 or later

3. Navigate to the desired project folder

4. Press `F5` to build and run, or use:
   ```bash
   dotnet run
   ```
